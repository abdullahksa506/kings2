# King of Thursday (ملك الخميس)

A premium web application to manage the official 2026 King of Thursday constitutional outings.

## Features
- Secure Login System
- Cycle and King Rotation Enforcements
- Deadline Tracking (Wednesdays 8PM / 10PM)
- Hidden voting and 5-Star Rating System
- Dean's Secret Administrative Dashboard
- Public Leaderboard for the active cycle

Built with Next.js, Tailwind CSS, and Firebase Firestore.
